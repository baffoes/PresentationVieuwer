README - PresentationVieuwer

-Beschrijving
Dit project is een applicatie voor het laden en weergeven van presentaties uit gecomprimeerde JSON EN XML-bestanden.
De gebruiker kan een bestand selecteren, het wordt uitgepakt, en de presentatie wordt geladen en weergegeven met behulp van een grafische interface.

-Functies
Bestanden selecteren: Gebruikers kunnen een bestand selecteren om te openen via een bestandkiezer.
Presentatie laden: Het geselecteerde bestand wordt uitgepakt en geladen om de inhoud van de presentatie weer te geven.
Navigatie door dia's: De gebruiker kan door de dia's bladeren, naar een specifieke dia gaan of de presentatie afsluiten.
Afsluiten van de applicatie: De applicatie wordt afgesloten en de gebruikte bronnen worden opgeruimd.

git clone https://github.com/baffoes/PresentationVieuwer.git
